### 3.2 Wave grammar (generate, don’t hand-wave 150)

```
contentLevel L → pick ChapterTemplate(L)
  for each slot in template:
    unit = pick from chapter unit pool (weighted)
    count = baseCount(L) * densMul
    spacing = max(0.04, baseSpacing / densMul)
    props = rollProps(L, rngWave)  // camo/lead/regrow/fort chance by chapter
  if L in bossSchedule: bossInsert
  if jackpotRoll(L, mode.jackpot): insert 1 JACKPOT
```

| Chapter | L range | Unit pool bias | Prop chances (base) |
|---------|---------|----------------|---------------------|
| Teach | 1–15 | green, ripe, gold | 0 |
| Open | 16–30 | gold, purple | camo 8%, lead 5% |
| Spike | 31–55 | purple, star, zebra, ceramic | camo 15%, lead 20%, regrow 8% |
| Craft | 56–90 | zebra, ceramic, mixed props | camo 22%, lead 25%, fort 10% |
| Endurance | 91–150 | all + freighter/titan schedule | stacked 30%+ |

**Boss schedule (contentLevel):** barge @ 30,40,60; freighter @ 50,70,90; titan @ 100,120,150 (±2).

**Mode dens/ban:** multiply counts and bounties via SYSTEMS-BIBLE — do not rewrite books per mode.

**Map independence:** same wave book; map only changes geometry (difficulty feel).

**Tour:** same book; every `tourEvery` waves change map, relocate towers.
